1. Download the source code using this link "https://github.com/Hrishith30/mars.git"
2. Next extract the zip file and navigate to the folder using command prompt.
3. Type "npm install" to install all the required dependencies.
4. After installing all the dependencies type "npm test" and hit enter this will run unit tests for the form validation logic using Jest.
5. Later try "npm start" to run the code.
